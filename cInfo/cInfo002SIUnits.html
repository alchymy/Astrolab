SI Units

The international system of units consists of a set of base units together with a set of prefixes.

There are the seven base units.  Each of these base units are dimensionally independent:

SI base units
Name Symbol Quantity
kilogram kg Mass
second s Time
metre m Length
ampere A Electrical current
kelvin K Temperature
mole mol Amount of substance
candela cd Luminous intensity

From these seven base units several other units can be derived.

In addition to the SI units there are also a set of non-SI units accepted for use with the SI system.

Unless otherwise stated only SI units are used in this application and apart from the base units only those used in the application will be described.

A prefix may be added to units to produce a multiple of the original unit.
All multiples are integer powers of ten. For example, kilo- denotes a multiple of a thousand and milli- denotes a multiple of a thousandth hence there are one thousand millimetres to the metre and one thousand metres to the kilometre.

NOTE 1: WHEREVER POSSIBLE NATIONAL PHYSICAL LABORATORY DEFINITIONS ARE USED FOR VALUES IN THIS APPLICATION AS A STANDARD, OTHERWISE THE SOURCE WILL BE NOTED.

NOTE 2: Planetary scientists often use 'kilometer' for length and 'bar' for pressure.  1 mile = 1.6 km.  1 bar = Earth's air at sea level  = 105 pascals or 1 pascal = 10-5 bar.  The centimeter-gram-second units can still be found widely in astronomy and astrophysics.  The table below gives some abbreviations and conversions.
