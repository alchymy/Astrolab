Data Sources on Galaxies


  • Data on Galaxies

There are many sources of data on galaxies to be found at different locations on the web.

These include databases such as:

LEDAS – http://ledas-www.star.le.ac.uk The Leicester Database and Archive Service (LEDAS). This is an extensive, multiwaveband set of databases and data, originally based around X-ray missions. You can browse and find information on individual sources by name or position.

NED – http://nedwww.ipac.caltech.edu The NASA/IPAC Extragalactic Database (NED). Contains information about extragalactic sources, including references for named sources; positions; background notes; photometry etc. extracted from published literature.

DRAGNs – http://www.jb.man.ac.uk/atlas/dragns.html Atlas of Double Radiosources Associated with Galactic Nuclei (DRAGN). Contains radio images for radio emitting galaxies with a double-lobed structure.

There are literature reference sources such as:

ADS – http://adsabs.harvard.edu/abstract_service.html The NASA Astrophysics Data System (ADS) Article Service. Search by author name; year or specific keywords. It is best to restrict the dates to later than about 1995, for more recent publications. This facility gives you access to abstracts and alsomore complete versions of published papers.
