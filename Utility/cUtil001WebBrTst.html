This page contains links to diagnostic tests for web browsers to test their compliance with web standards.  These standards are important to ensure all browsers work the same way and that code does not have to be re-written to cope with individual browsers from different sources!!

The ACID 3 Test from the Web Standards Project is the latest test for your browser's compatibility with current standards, for more information on this test read the Wikipedia entry

The above test is based on default browser settings.

Load SVG file test - core Web Browser must support Vector Graphics files (.svg) without plugins, this application may require vector graphics.

For more information on standards visit the Web Standards Project  or the World Wide Web Consortium

Other 'older' ACID tests from the Web Standards Project can be found at this link http://www.acidtests.org/

For MathML rendering try Maths 1 or Maths 2 (this is still experimental).
