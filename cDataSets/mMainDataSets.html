This page contains various Data Sets for reference and research

NOTE: you must have applications installed on your platform which can open and read the file formats attached to each link below.

Open Office and NEO Office (Mac) are the free recommended tools for reading all file formats.  Data items highlighted in green have been verified.
Data items highlighted in YELLOW have yet to be processed and verified into a standard format.

Where possible all data sets are supplied in the three file formats shown for Periodic Table.

The folder cDataSets can be installed separately in order to keep the total size of the application down. The cDataSets folder can be downloaded separately and simply copied into the main Astrolab application folder.
